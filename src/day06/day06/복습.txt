[화]
    연산자 : 연산/계산 할때 사용되는 문자/기호
        - 문자 연결 연산자 :  +
        - 산술 연산자 : + - * ** / // %
        - 비교 연산자 : > < >= <= == !=
        - 논리 연산자 : and or not
        - 대입 , 복합대입연산자
            = += -= *= **= /= //= %=
        - 삼항 연산자
            True실행문 if 조건문 else False실행문
[목]
    조건문 : 조건/경우 에 따른 서로(True/False) 다른 실행문 처리
        - 형태
            1. if 조건 : True실행문
            2.  if 조건 : True실행문
                else : False실행문
            3.  if 조건1 : True실행문
                elif 조건2 : True실행문2
                elif 조건3 : True실행문3
                else : False실행문
[금]
    반복문 : 조건/경우 에 따른 실행문 을 반복 처리
        - 1. for 반복변수명 in range(   )
            - * range(   )  함수
                range( 수 ) : 0부터 수 전까지 1씩 증가하면서 반복변수 대입 반복 처리
                range( 수1 , 수2 ) : 수1 부터 수2 전까지 1씩 증가하면서 반복변수 대입 반복 처리
                range( 수1 , 수2 , 수3 ) 수1 부터 수2 전까지 수3 씩 증감하면서 반복변수 대입 반복 처리

        - 키워드
            continue    : 현재 반복을 중단하고 다음 반복으로 이동. [ 가장 가까운 반복문으로 이동 ]
            break       : 현재 반복을 중단하고 전체 반복문을 종료. [ 가장 가까운 반복문 종료 ]
