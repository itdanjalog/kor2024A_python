- (데이터/자료) 저장 개념
    1. 변수
        - 변하는 값 , 하나의 데이터 저장/기억
        - 데이터 대한 상징 적인 이름*
    2. 데이터
        - 타입 / 데이터 들을 특정 기준 으로 분류
            - 기본 타입 : 정수(int) , 실수(float) , 문자열(str) , boolean(불리언)
            - 참조 타입 : 리스트[] , 튜플() , 딕셔너리{}
    3.
    변수명 = 정수데이터
    변수명 = 실수데이터
    변수명 = 문자열데이터
    변수명 = 불리언데이터
    변수명 = [ value , value , value , value , value , value , value  ]
    변수명 = ( value , value , value , value , value , value , value )
